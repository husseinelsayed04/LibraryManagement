# 📚 Library Management System

A **simple Java Library Management System** with a **console interface** that allows librarians to manage books, magazines, and journals efficiently.

---

## ✨ Features

- Add **Books**, **Magazines**, and **Journals**
- View **available library items**
- **Lend** items to users
- **Receive** items from users
- **Search** items by type and title
- Generate **borrowed items report**
- Generate **summary report** of all library items
- **Librarian login authentication**

---

## 🛠 Technologies Used


- Java
- Java Collections Framework (ArrayList, HashMap)
- Object-Oriented Programming (OOP) Principles (Encapsulation, Inheritance, Polymorphism)
